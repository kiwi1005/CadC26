# Step7C-real-B: Official-Metric-Directed Real Edit Targeting

## Plan

Step7C-real-B follows Step7C-real-A's decision:

```text
refine_real_edit_generators
```

Step7C-real-A showed that route-aware real placement edits are safe enough to
continue sidecar research, but its official-like metric signal was too sparse:
only one candidate improved official-like cost. Step7C-real-B keeps the same
sidecar and routing boundaries, but changes the generator objective from generic
nudges to deterministic metric-directed targeting.

## Scope

- sidecar only
- no contest runtime integration
- no finalizer semantics change
- no RL / trained ML
- no multi-iteration search
- no hard discard gate from locality routing
- no silent dropping of regional, macro, or global candidates
- no sending global candidates to bounded-local repair
- original-inclusive Pareto reporting

## Implemented strategies

`src/puzzleplace/alternatives/metric_directed_edits.py` emits one real candidate
per case per deterministic strategy:

- `original_layout`
- `hpwl_directed_local_nudge`
- `bbox_shrink_nudge`
- `slack_fit_insertion`
- `balanced_swap`
- `regional_metric_directed_shift`
- `macro_closure_metric_move`
- `legacy_step7g_global_report`

Each non-global strategy internally probes bounded deterministic alternatives,
filters official-like hard feasibility, then selects the best official-like cost
delta from a small shortlist. The emitted artifact remains one-step and
sidecar-only; internal probes are not iterative search.

## Run result

Generated on the Step7G/H/C-thin/real-A case set:

- Real FloorSet cases: 8.
- Candidates: 64 = 8 strategies × 8 cases.
- Routes: local 43, regional 5, macro 8, global 8.
- `non_global_candidate_rate`: 0.875.
- `invalid_local_attempt_rate`: 0.0.
- `official_like_hard_feasible_rate`: 0.875.
- Original-inclusive Pareto front non-empty: 8 / 8 cases.
- Official-like cost improving candidates: 3, up from Step7C-real-A's 1.
- Improvement density among feasible candidates: 0.1429.
- Official-like cost improvement density among feasible candidates: 0.0536.
- Best strategies by official-like improvement count:
  - `slack_fit_insertion`: 2.
  - `hpwl_directed_local_nudge`: 1.
- Regional/macro candidates are preserved and hard-feasible, but still have low
  non-noop/metric improvement density.

## Decision options

- `promote_to_step7c_multi_iteration_sidecar`
- `refine_metric_directed_generators`
- `build_route_specific_legalizers`
- `pivot_to_coarse_region_planner`
- `pivot_to_macro_level_move_generator`
- `revisit_official_metric_model`
- `inconclusive_due_to_metric_signal`

## Decision

```text
refine_metric_directed_generators
```

Reason: official-metric-directed targeting improved the main bottleneck versus
Step7C-real-A (`official_like_cost_improving_count` rose from 1 to 3) while
preserving route safety, hard feasibility, and original-inclusive Pareto
coverage. The signal is still too sparse for multi-iteration Step7C promotion:
regional/macro strategies are mostly no-op or non-improving, and official-like
cost improvement density remains low. The next sidecar work should refine
metric-directed generators and/or route-specific construction before promotion.
