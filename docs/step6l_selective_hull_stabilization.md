# Step6L Selective Hull Stabilization with Shape / Group Probes

## Scope

Step6L is a research sidecar. It does not change contest runtime, finalizer,
rollout, or scorer behavior.

Inputs:

- `artifacts/research/step6j_boundary_hull_ownership.json`
- `artifacts/research/step6k_boundary_failure_attribution.json`

Step6K identified overlapping boundary failure causes, so Step6L evaluates
alternatives before promoting any one fix.

## Implemented diagnostics

1. **Hull drift metrics**
   - Compare predicted compact hull against final bbox.
   - Record per-edge drift and final bbox owner roles.

2. **Attribution co-occurrence**
   - Count `failure_type x role_flag`, `failure_type x edge`,
     `failure_type x boundary_type`, and external-ratio buckets.

3. **Selective hull stealing guard audit**
   - Audit non-boundary/regular hull owners.
   - Keep exceptions for fixed/preplaced, terminal-heavy, low expansion, and no
     available boundary owner.

4. **Compaction alternatives**
   - Evaluate original, simple compaction, and edge-aware compaction.
   - Reject hard-infeasible or frame-protruding candidates.

5. **Shape / MIB / group probes**
   - Probe only unsatisfied boundary blocks, hull stealers, and role-conflict
     targets.
   - Emit MIB compatibility and group-template diagnostics without globally
     enabling new search freedom.

6. **Alternative selection**
   - Lexicographic selection: boundary satisfaction, bbox, HPWL, soft
     violations, role-conflict churn.
